What's the kind of program is that?

- This program, reads the current exchange rate against pln (Polish zloty)

How to correctly use this program? 

1. In Form2, you write the currency, for example: Euro - eur, like abbreviation. Now will be create a file.txt (eur.txt). 
2. Next step, you will see in section: "Podana lista walut:" picked currency.
3. If you want to read the exchange, pick the name of currency in section: "Z którego pliku wyliczyc srednia? (bez końcówki .txt!)"
4. Now, click the button "Uśrednij!" 
5. tThe result will appear in section: "Podany usredniony kurs:".
